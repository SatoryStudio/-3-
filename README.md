# Filament ERP

Локальная ERP-система для заказов, 3D-печати, филамента и расчёта прибыли.

## Запуск

1. Установить Node.js 22+.
2. Создать `.env.local` по образцу `.env.example`.
3. Установить зависимости и запустить приложение:

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:3000`. При первом запуске появится мастер создания администратора.

`APP_URL` задаёт фактический адрес установки и используется для автоматического формирования Google OAuth callback.

## Хранилище

Рабочие данные создаются в каталоге `data/` в отдельных XLSX-файлах. Не редактируйте их вручную во время работы приложения.

Каждая запись:

- проходит через серверные доменные сервисы;
- блокирует общее хранилище от конкурентной записи;
- создаёт backup затрагиваемых файлов;
- записывается во временный каталог и проверяется;
- атомарно заменяет рабочие файлы.

Резервные копии находятся в `data/backups/`.

## Интеграции и worker

Google Sheets, Яндекс Маркет и Ozon подключаются в разделе «Настройки». Секреты шифруются AES-256-GCM и сохраняются в `data/integration_settings.xlsx`; API показывает только маски.

Google Sheets является основным интерфейсом товаров и катушек. Постоянные таблицы открываются и синхронизируются из раздела «Таблицы».

Worker работает внутри Next.js-процесса. На вкладке «Worker» можно изменить интервалы, запустить, остановить, перезапустить или выполнить синхронизацию немедленно.

На вкладке «Принтеры» можно добавить ручной принтер или сохранить реквизиты Bambu Lab. Ручной принтер назначается заданию в очереди; при завершении вводится фактический расход, после чего остаток катушки и прибыль пересчитываются сервером.

